# 🤖 Artificial Intelligence Project (Beginner ML Model)

This is a simple Machine Learning project using Python and scikit-learn.

---

## 📌 Features
- Linear Regression model
- Simple dataset
- Prediction system
- Clean modular structure

---

## 🧠 What it does
The model learns a simple pattern:

X → Y relationship (2x rule)

Example:
1 → 2  
2 → 4  
3 → 6  

Then it predicts new values.
